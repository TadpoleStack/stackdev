## Tadpole技术栈
##### 这是一个基于vuepress搭建的静态技术文档，随着vuepress的更新一直在不断的完善中...
- 引入element-ui进行组建的运用
- 通过引入第三方库jquery + swiper + three.js简单的效果制作
- 通过valine进行评论模块的制作
- 设置全局的背景音乐
- axios进行数据的请求
- moment进行时格式化
- 设置自定义域名，通过自定义travis一键发布到GitHub、腾讯云开发者平台（coding）、码云
